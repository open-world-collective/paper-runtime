# AI Provider Policy

Open World Collective only uses AI providers that take an explicit, non-negotiable public and contractual stance against surveillance and harm.

## Minimum requirements

1. Explicit prohibition of mass domestic surveillance, biometric tracking/profiling, and social scoring.
2. Explicit prohibition of fully autonomous weaponization and weapon-use enablement.
3. No carve-outs that allow these prohibited uses through national-security or intelligence exceptions.
4. If a government customer requires removal of these safeguards, the provider must refuse.

## Provider status (as of March 3, 2026)

- Approved: Anthropic
- Not approved: OpenAI

## Governance

- Any exception or provider change requires an explicit governance decision.
- If a provider weakens these protections, usage is suspended immediately.
