# Code of Conduct

We expect respectful, professional collaboration focused on technical outcomes.

## Expected behavior

- Assume good intent, ask clarifying questions, and stay specific.
- Critique code and process, not people.
- Document decisions and unresolved risks.

## Unacceptable behavior

- Harassment, discrimination, threats, or doxxing
- Repeated hostile communication
- Deliberate sabotage of infrastructure or release process

## Reporting

Report concerns to maintainers through private channels listed in repo settings.

## Enforcement

Maintainers may remove content, restrict participation, or block access based on severity and pattern.
